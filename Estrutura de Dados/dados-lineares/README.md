# Estruturas de Dados Lineares

Este módulo apresenta a implementação das principais estruturas de dados lineares utilizando JavaScript.

## Estruturas implementadas

- Vetores (Arrays)
- Lista simplesmente encadeada
- Lista duplamente encadeada
- Pilha (Stack)
- Fila (Queue)
- Fila circular

## Conceitos

### Vetores
Estrutura que armazena elementos de forma contígua na memória e permite acesso rápido por índice.

### Listas Encadeadas
Estrutura composta por nós conectados por ponteiros.

Tipos implementados:
- Lista simplesmente encadeada
- Lista duplamente encadeada

### Pilha
Estrutura que segue o princípio **LIFO (Last In First Out)**.

Operações:
- push
- pop
- peek

### Fila
Estrutura que segue o princípio **FIFO (First In First Out)**.

Operações:
- enqueue
- dequeue

## Linguagem

Todos os exemplos foram implementados em **JavaScript**.
